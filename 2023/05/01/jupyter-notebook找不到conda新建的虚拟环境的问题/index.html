Title

jupyter notebook找不到conda新建的虚拟环境的问题

发表于2023-05-01|更新于2023-05-01
|阅读量:

问题描述

在anaconda的命令行窗口用conda命令创建了虚拟环境cvenv，当切换到该虚拟环境，通过命令行打开jupyter notebook，在jupyter notebook中打开.ipynb文件时，找不到conda创建的虚拟环境的内核。

解决方案：将虚拟环境添加到jupyter notebook

  1. 切换到需要添加的虚拟环境中

  2. 安装ipykernel包
    pip install --user ipykernel

  3. 将虚拟环境添加到jupyter
    python -m ipykernel install --user --name=xxx

    如果输出
    Installed kernelspec xxx in /home/user/.local/share/jupyter/kernels/mypytorch
    说明操作成功

    再次打开jupyter notebook就可以使用新加的这个虚拟环境了

关于jupyter notebook虚拟环境的一些其它操作

  1. 查看jupyter notebook中有哪些虚拟环境kernel
    jupyter kernelspec list
  2. 删除指定的kernel
    jupyter kernelspec uninstall xxx
